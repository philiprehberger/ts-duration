# Changelog

## 0.1.0 (2026-03-15)

- Initial release
- Parse human-readable duration strings (`2h 30m 15s`)
- Parse ISO 8601 duration strings (`PT2H30M15S`)
- Format to human, short, and ISO representations
- Duration arithmetic (add, subtract, multiply, divide)
- Comparison methods
- `Duration.between()` for measuring time between dates
